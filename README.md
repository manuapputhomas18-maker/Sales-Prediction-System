# Sales Prediction and Analysis System

## Overview
This project uses MySQL, Python, Pandas, Matplotlib, and Machine Learning to analyze sales data and predict future sales.

## Technologies Used
- Python
- MySQL
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Connects to MySQL database
- Retrieves sales data using SQL queries
- Displays sales reports
- Visualizes sales trends using charts
- Predicts future sales using Linear Regression
- User input-based sales prediction

## Machine Learning Algorithm
- Linear Regression

## Skills Demonstrated
- SQL
- Data Analysis
- Data Visualization
- Machine Learning
- Python Programming
